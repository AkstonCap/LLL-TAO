Detailed Description

Defines the public key of a key pool entry. The KeyPoolEntry is written to the wallet database to store a pre-generated pool of available public keys for use by the wallet. The private keys corresponding to public keys in the key pool are not included.

These key pool entries can then be read from the database as new keys are needed for use by the wallet.

Database key is pool<nPool> where nPool is the pool entry ID number
