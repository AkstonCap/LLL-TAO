Detailed Description

Base

Base class for all JSON based API's.

Instances of JSONAPIBase derivations must be registered with the JSONAPINode processing the HTTP requests. This class holds a map of JSONAPIMethod instances that in turn perform the processing for each API method request.

Execute

Handles the processing of the requested method. Derivations should implement this to lookup the requested method in the mapFunctions map and pass the processing on. Derivations should also form the response JSON according to the API specification

SanitizeParams

Allows derived API's to check the values in the parameters array for the method being called. The return json contains the sanitized parameter values, which derived implementations might convert to the correct type for the method being called.

Parameters

[in]jsonParamsThe json array of parameters being passed to this method.
