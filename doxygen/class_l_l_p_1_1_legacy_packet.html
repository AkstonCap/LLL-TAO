Detailed Description

LegacyPacket

Class to handle sending and receiving of More Complese Message LLP Packets.

Components of a Message LLP Packet. BYTE 0 - 4 : Start BYTE 5 - 17 : Message BYTE 18 - 22 : Size BYTE 23 - 26 : Checksum BYTE 26 - X : Data
