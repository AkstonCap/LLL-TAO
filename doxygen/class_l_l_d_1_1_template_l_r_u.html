Detailed Description

template<typename KeyType, typename DataType>
class LLD::TemplateLRU< KeyType, DataType >

TemplateLRU

LRU - Least Recently Used. This class is responsible for holding data that is partially processed. This class has no types, all objects are in binary forms.
