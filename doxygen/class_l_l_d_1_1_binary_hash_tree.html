Detailed Description

BinaryHashTree

This class is responsible for managing the keys to the sector database.

It contains a Binary Hash Tree with a search complexity of O(log n) It uses files as leafs or nodes when collisions are found in buckets making it a hybrid hashmap and binary search tree
