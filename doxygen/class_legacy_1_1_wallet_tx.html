Detailed Description

A transaction with additional information relevant to the owner and the owner's wallet which owns it. It includes any unrecorded previoius transactions needed to link it back to the block chain.

Database key is tx<hash> where hash is the transaction hash

◆ vfSpent

std::vector<bool> Legacy::WalletTx::vfSpent

char vector with true/false values indicating spent outputs. Vector index matches output index value. Outputs with index value > vfSpent.size() are considered unspent (not marked as spent).

Not serialized directly. Values are copied into an entry in mapValue using the "spent" key for serialization/deserialization.
