Detailed Description

Base58-encoded Nexus Addresses.

Public-key-hash-addresses have version 42 (or 111 testnet). Script-hash-addresses have version 104 (or 196 testnet). The data vector contains SK256(cscript), where cscript is the serialized redemption script.
