Detailed Description

DDOS_Score

Class that tracks DDOS attempts on LLP Servers.

Uses a Timer to calculate Request Score [rScore] and Connection Score [cScore] as a unit of Score / Second. Pointer stored by Connection class and Server Listener DDOS_MAP.
